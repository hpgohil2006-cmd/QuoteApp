# 📱 Quote Generator App (Flutter)

A beautifully designed **Quote Generator Mobile Application** built using **Flutter**, featuring a modern UI with smooth animations, glassmorphism design, and interactive functionality. This app allows users to generate inspirational quotes, store them, and view them anytime in a dedicated section.

---

## 🚀 Features

✨ **Random Quote Generator**

* Generates motivational quotes instantly with a single tap

🎨 **Modern UI Design**

* Gradient background
* Glassmorphism (blur) quote card
* Smooth fade animation

📂 **Stored Quotes Management**

* Automatically stores generated quotes
* Prevents duplicate entries
* Dedicated screen to view all stored quotes

📜 **Quote Viewer Screen**

* Displays all saved quotes in a clean list layout

⚡ **Responsive & Smooth Performance**

* Optimized Flutter UI for fast and smooth experience

---

## 🛠️ Technologies Used

* **Flutter** (UI Development)
* **Dart** (Programming Language)
* **Material Design Components**

---

## 📱 How It Works

1. Launch the app
2. Tap on **"New Quote"** button
3. A random quote is displayed with animation
4. The quote is automatically saved
5. Tap **"View Stored Quotes"** to see all generated quotes
